# AI Resume Checker using Make.com

This project is an AI-based resume checker that automates resume collection, analysis, and email reporting using a Telegram bot and Make.com.

Users interact with a [Telegram bot](https://t.me/Fayaz_Resume_checker_bot)  by sending a "Hi" message. The bot requests a resume link from the user. Once submitted, the resume is processed in the backend using Make.com, where key details are extracted and analyzed. After processing, a resume analysis report is automatically sent to the email address found in the resume.
